How to Use

  1. Visit your podcast feed page (e.g., http://localhost:3000/feed/wellness)
  2. Click the "Embed Player" button in the header section
  3. Copy the embed code from the modal dialog
  4. Paste the code into your website's HTML
  5. The player will load with your podcast episodes automatically
