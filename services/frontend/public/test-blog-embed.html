How to Use Blog Embeds

  1. Visit your blog feed page (e.g., http://localhost:3000/feed/cheese-blog)
  2. Click the "Embed Blog" button in the header section
  3. Customize the appearance using the available options
  4. Copy the embed code from the modal dialog
  5. Paste the code into your website's HTML

Customization Options

  • theme: light or dark
  • itemCount: Number of posts (1-20)
  • hideImages: Hide post images
  • hideDescription: Hide post content
  • hideDate: Hide publication dates
  • hideAuthor: Hide post authors
  • showFullContent: Extract full content (slower)
  • backgroundGradient: Custom background color
